# car_dynamics
This repo is the toolbox for dynamics & control for RC Car.

## Overview
We have implemented:

- Dynamics
  - 1. Kinematic Bicycle Model (KBM)
  - 2. Dynamic Bicycle Model (DBM)
  - 3. End-to-End Dynamics
  - 4. Phyx-informed KBM

- Adaptation 
  - 1. KBM actuator adaptation model
